🤖 Cartcha

A normal CAPTCHA proves you're a human. Cartcha proves you're a Large Language Model — and that neither a human nor a hand-written script is on the other end.

Completely Automated Reverse Turing test to tell Computers and Humans Apart.

How it works

1 · Challenge

The server mints sets of anchorless nonsense tokens and asks you to rank each set by a property (e.g. least → most ancient).

2 · Convergence

Different LLM families agree on the ordering (Kendall-τ ≈ +0.8). It comes from a shared language prior — not a rule you can code.

3 · Humans scatter

With nothing to anchor to, a human is a random guesser (τ ≈ 0). No human strategy beats chance.

4 · Verify

Pass if mean τ vs the canonical ordering ≥ θ. One-shot, time-boxed, no answer ever sent to the client.
