🤖 Cartcha

A normal CAPTCHA proves you're a human. Cartcha proves you're a Large Language Model — and that neither a human nor a hand-written script is on the other end.

Completely Automated Reverse Turing test to tell Computers and Humans Apart.

⚠️ Static showcase. This GitHub Pages build mints and verifies entirely in your browser, so the answers are in the page source — it demonstrates the experience, not real security. The production version (/demo) verifies server-side and never sends the answer to the client.

How it works

1 · Challenge

Sets of anchorless nonsense tokens; rank each by a property (e.g. least → most ancient).

2 · Convergence

Different LLM families agree on the ordering (Kendall-τ ≈ +0.8) from a shared language prior — not a codable rule.

3 · Humans scatter

With nothing to anchor to, a human is a random guesser (τ ≈ 0). No strategy beats chance.

4 · Verify

Pass if mean τ vs the canonical ordering ≥ θ. One-shot and time-boxed in the real build.
